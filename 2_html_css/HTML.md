# HTML

## 구조

### head

```
<!--        이것은 주석이다-->       # ctrl + /  주석처리하기
<!--        이것도 주석처리하자-->
```

* <meta>

  : 클로징 필요 없음

  : charset = "utf-8"

* <link>

  : 클로징 필요 없음

* <title>Heading</title>

  : 클로징 자동생성

### body

* <h1> 가장 큰 제목 </h1>

* Ctrl + d 는 라인 카피

## 터미널에서 사용하는 것

``` sh
cp templete.html heading.html 
# templete.html을 복사하여 heading.html이라는 이름으로 만들기
mv block_inline.html div.html
# 파일 이름 바꾸기
```
