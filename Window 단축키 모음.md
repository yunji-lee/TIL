# Window 단축키 모음 



##  Window 

* 창 최소화 

  `Win + M`

* 창 최대화

  `Shif + Win + M`

  

* 시스템 잠그기

  `Win + L`

* 시스템 도구 사용하기

  `Win + X`

* 작업보기 화면

  `Win + tab`

  

* 새 가상 바탕화면 만들기 

  `Win + Ctrl + D`

* 새 가상 화면 지우기 

  `Win + Ctrl F4`

* 만들어진 바탕화면 사이를 전환

  `Win + Ctrl + 왼쪽/오른쪽 `

* 바탕화면 보기 

  `Win + D`
  
  

## Alt 

* **바탕화면 전체에서 열린 프로그램 사이를 전환 **

  `Alt + tab`

* 뒤로

  `Alt + <`

* 앞으로

  `Alt + >`



## Ctrl 

* 글자 말고 단어 전체 지우기

  `Ctrl + Del or Backspace`

* 파일 열기 화면 

  `Ctrl + O`

  

* 현재 보고 있는 탭 닫기

  `Ctrl + F4` , `Ctrl + W`

* **현재 창에서 열린 프로그램 사이를 전환**

  `Ctrl + Tab`

* 창에서 새 탭 열기

  `Ctrl + T`

  

* 페이지 완전히 새로 불러오기

  `Ctrl + F5`

  

*  화면 확대

  `Ctrl + +`

* 화면 축소

  `Ctrl + - `

* 표준 비율로 돌아가기 

  `Ctrl + 0`

  

* 커서 주소 표시줄로 이동시키기 

  `Ctrl + L`

* 주소 표시줄에서 'www', 'com' 자동 생성

  `키워드 검색 + Ctrl+Enter`

  

* 방문기록 열기

  `Ctrl +  H`

* 다운로드 기록 보기

  `Ctrl + J`

* 현재 사이트 북마크 추가

  `Ctrl + D`

* 방문 기록 삭제 

  `Ctrl + Shift + Del`
